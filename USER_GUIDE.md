# WebCat 使用指南

## 📖 简介

WebCat 是一款专为 Cocos Creator 网页游戏设计的资源抓取工具，可以帮助开发者学习和研究游戏资源。

## ⚠️ 重要提示

**本工具仅供个人学习研究使用，请勿用于商业目的或侵犯他人版权。**

## 🚀 安装和启动

### 方式一：源码运行（推荐开发者）

1. **安装 Node.js**
   - 下载地址：https://nodejs.org/
   - 要求版本：>= 16.x

2. **克隆或下载项目**
   ```bash
   cd webCat
   ```

3. **安装依赖**
   ```bash
   npm install
   ```

4. **启动应用**
   ```bash
   npm run electron:dev
   ```

### 方式二：安装包运行

1. 下载最新版本的安装包
2. 双击运行安装程序
3. 安装完成后启动 WebCat

## 📝 使用步骤

### 第一步：输入目标网址

1. 在顶部输入框中输入要抓取的网页游戏地址
2. 例如：`https://example.com/cocos-game/`
3. 点击"开始抓取"按钮

![URL输入](docs/images/step1.png)

### 第二步：等待资源加载

- 系统会自动打开浏览器窗口
- 等待游戏资源完全加载
- 资源列表会实时更新
- 可以在左侧看到各类资源的统计

![资源加载](docs/images/step2.png)

### 第三步：筛选资源

在左侧筛选面板中：

1. **按类型筛选**
   - ✅ 勾选想要的资源类型
   - 📷 图片、🎵 音频、🎬 视频等
   - 🎮 Cocos 专有资源格式

2. **关键词搜索**
   - 输入 URL 关键词进行精确搜索
   - 支持实时过滤

3. **快捷操作**
   - 点击"全选"选择所有类型
   - 点击"清空"取消所有选择

![资源筛选](docs/images/step3.png)

### 第四步：选择要下载的资源

在右侧资源列表中：

1. 勾选顶部的"全选"框可以选中所有资源
2. 或逐个勾选需要的资源文件
3. 可以点击 📋 按钮复制资源链接
4. 底部状态栏会显示已选择的数量

![选择资源](docs/images/step4.png)

### 第五步：下载资源

1. 点击左侧"📁 选择保存目录"按钮
2. 选择资源保存的文件夹
3. 点击"⬇️ 开始下载"按钮
4. 等待下载完成

![下载资源](docs/images/step5.png)

### 第六步：查看下载结果

下载完成后，文件会按类型分类保存：

```
你选择的目录/
├── image/          # 图片资源
├── audio/          # 音频资源
├── video/          # 视频资源
├── script/         # 脚本文件
├── cocos/          # Cocos Creator 资源
└── ...
```

## 🎯 功能详解

### 1. 资源类型识别

WebCat 支持识别以下资源类型：

| 图标 | 类型 | 说明 | 文件扩展名 |
|-----|------|------|-----------|
| 📷 | 图片 | 游戏图片、精灵图 | jpg, png, webp, svg |
| 🎵 | 音频 | 背景音乐、音效 | mp3, wav, ogg, m4a |
| 🎬 | 视频 | 游戏视频、过场动画 | mp4, webm, mov |
| 📝 | 脚本 | JavaScript 代码 | js, jsc, ts |
| 🎨 | 样式 | CSS 样式表 | css |
| 🔤 | 字体 | 字体文件 | woff, ttf, otf |
| 📋 | JSON | 配置数据 | json |
| 🎮 | Cocos | Cocos Creator 专有格式 | plist, atlas, bin, prefab |
| 📦 | 其他 | 其他资源文件 | - |

### 2. 智能过滤

- **类型过滤**：只显示你感兴趣的资源类型
- **关键词搜索**：快速找到特定文件
- **实时统计**：显示每种类型的资源数量

### 3. 批量下载

- 支持一次性下载多个资源
- 自动按类型分类保存
- 显示下载进度
- 失败自动重试（开发中）

### 4. 资源预览（规划中）

- 图片资源缩略图预览
- 音频资源在线播放
- JSON 数据格式化显示

## 💡 使用技巧

### 技巧 1：等待资源完全加载

- 游戏可能会延迟加载某些资源
- 建议在游戏中点击各个功能触发资源加载
- 抓取完成前不要关闭浏览器窗口

### 技巧 2：利用关键词搜索

如果资源很多，可以使用关键词快速定位：

- 搜索 `sprite` 查找精灵图
- 搜索 `bg` 或 `background` 查找背景图
- 搜索 `sound` 或 `audio` 查找音频

### 技巧 3：选择合适的保存目录

- 建议为每个游戏创建单独的文件夹
- 目录名可以包含游戏名称和日期
- 例如：`D:\GameResources\MyGame_2024-01-01\`

### 技巧 4：查看资源详情

- 鼠标悬停在资源 URL 上可以看到完整地址
- 文件大小显示在类型旁边
- 可以通过复制链接直接在浏览器中查看

## ❓ 常见问题

### Q1: 为什么有些资源没有抓取到？

**可能原因：**
- 资源是动态加载的，需要在游戏中触发
- 资源使用了特殊的加载方式
- 网络请求被拦截或加密

**解决方法：**
- 在游戏中多进行操作，触发更多资源加载
- 等待更长时间再停止抓取
- 尝试刷新页面重新抓取

### Q2: 下载速度很慢怎么办？

**可能原因：**
- 资源服务器在国外
- 网络状况不佳
- 同时下载的文件过多

**解决方法：**
- 分批下载，不要一次选择太多资源
- 使用网络加速工具
- 选择网络状况好的时间段

### Q3: Cocos Creator 资源如何使用？

Cocos Creator 的打包资源需要特殊处理：

- `.jsc` 文件是编译后的脚本，难以直接阅读
- `.plist` 和 `.atlas` 是精灵图集信息
- `.bin` 文件可能包含场景或预制体数据

建议使用专门的 Cocos Creator 工具来解析这些资源。

### Q4: 下载的图片无法打开？

**检查项：**
- 文件扩展名是否正确
- 文件大小是否为 0
- 是否需要特殊的查看器

### Q5: 抓取过程中浏览器崩溃

**可能原因：**
- 内存不足
- 游戏资源过大
- 浏览器版本问题

**解决方法：**
- 关闭其他程序释放内存
- 更新 Chrome 或 Edge 浏览器
- 重启应用重试
